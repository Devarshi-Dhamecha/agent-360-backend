API Response System - Folder Structure
=====================================

Quick Import Reference:
======================

# Responses
from core.api.responses import APIResponse, SuccessResponse, ErrorResponse

# Exceptions
from core.api.exceptions import (
    ResourceNotFoundException,
    ValidationException,
    UnauthorizedException,
    custom_exception_handler
)

# Serializers
from core.api.serializers import (
    BaseModelSerializer,
    AuditedSerializer,
    ListRequestSerializer
)

# Utils
from core.api.utils import StandardPagination


Usage Examples:
==============

1. Success Response:
   return SuccessResponse.retrieve(data=serializer.data)

2. Error Response:
   raise ResourceNotFoundException("User", user_id)

3. Pagination:
   paginator = StandardPagination()
   return paginator.get_paginated_response(serializer.data)

4. Validation:
   from core.api.utils import validate_salesforce_id
   validate_salesforce_id(id_value)


Response Format:
===============

Success:
{
    "success": true,
    "message": "...",
    "data": {...},
    "meta": {...}
}

Error:
{
    "success": false,
    "message": "...",
    "error_code": "...",
    "errors": [...]
}
